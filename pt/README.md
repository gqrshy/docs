### CobbleRanked

Um sistema abrangente de batalhas ranqueadas competitivas para servidores Cobblemon.

**Recursos:**
- 🏆 Sistema de busca de partidas baseado em Elo
- ⚔️ Múltiplos formatos de batalha (Singles, Doubles)
- 📊 Tabelas de classificação sazonais com rotação automática
- 🚫 Restrições personalizáveis de Pokémon/movimentos/habilidades
- 🌍 Suporte multi-servidor (MySQL + Redis)
