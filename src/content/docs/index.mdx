## Quick Start

```bash
# 1. Install dependencies (GashiLibs, MailLib)
# 2. Place CobbleRanked.jar in mods/
# 3. Start server
# 4. Test with /ranked command
```

| Requirement | Version |
|-------------|---------|
| Minecraft | 1.21.1 |
| Cobblemon | 1.7.1+ |
| GashiLibs | 1.0.3+ |
| MailLib | 1.0.1+ |
