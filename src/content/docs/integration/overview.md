CobbleRanked integrates with several mods to provide additional features.

## Required Dependencies

| Mod | Purpose |
|-----|---------|
| [GashiLibs](/integration/gashilibs/) | Shared libraries (DB drivers, config) |
| [MailLib](/integration/maillib/) | Reward delivery system |

## Optional Integrations

| Mod | Purpose |
|-----|---------|
| [LuckPerms](/integration/luckperms/) | Permission management |
| [Placeholder API](/integration/placeholders/) | Dynamic text placeholders |

## Integration Guides

- [GashiLibs](/integration/gashilibs/) - Required shared libraries
- [MailLib](/integration/maillib/) - Reward delivery via mailbox
- [LuckPerms](/integration/luckperms/) - Permission setup
- [Placeholders](/integration/placeholders/) - Using placeholders in other mods
