## Before You Start

Make sure you have:

| Requirement | Version |
|-------------|---------|
| Minecraft | 1.21.1 |
| Fabric Loader | 0.17.2+ |
| Fabric API | 0.116.6+ |
| Fabric Language Kotlin | 1.13.6+ |
| Cobblemon | 1.7.1+ |
| GashiLibs | 1.0.3+ |
| MailLib | 1.0.1+ |

See [Requirements](/docs/cobbleranked/getting-started/requirements/) for download links.

## 5-Minute Setup

1. **Download** all required mods
2. **Place** them in your `mods/` folder
3. **Start** the server (configs auto-generate)
4. **Test** with `/ranked` command

That's it! CobbleRanked works out of the box with sensible defaults.
