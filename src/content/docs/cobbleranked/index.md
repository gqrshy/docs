## Features

### Competitive Matchmaking
| Feature | Description |
|---------|-------------|
| **ELO System** | Smart skill-based matchmaking ensures fair, competitive battles. Watch your rating climb as you dominate! |
| **Battle Formats** | Singles, Doubles, and Triples - master them all! Each format has its own leaderboard and meta. |
| **Season System** | Competitive seasons with exclusive rewards and tier promotions. Earn your place among the elite! |
| **Leaderboards** | Real-time global and per-format rankings. See where you stand against the best trainers. |

### Advanced Features
| Feature | Description |
|---------|-------------|
| **Team Validation** | Configurable Pokemon, move, ability, and item blacklists. Create your perfect competitive ruleset. |
| **Battle Camera** | Cinematic battle experience with dynamic camera angles. Feel like you're in a real Pokemon battle! |
| **Cross-Server** | Connect multiple servers via Redis for unified rankings and queue. Perfect for large networks! |
| **Web API** | Sync leaderboards and usage stats to your website in real-time. [View Live Demo](/demo/leaderboard/) |
