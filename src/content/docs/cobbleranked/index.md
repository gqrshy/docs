A competitive ranked battle system for Cobblemon servers with ELO-based matchmaking, seasons, leaderboards, and rewards.

## Features

- **ELO Matchmaking** - Fair matches based on skill rating
- **Battle Formats** - Singles, Doubles, Triples support
- **Season System** - Competitive seasons with rewards
- **Leaderboards** - Global and per-format rankings
- **Team Validation** - Pokemon/move/ability/item blacklists
- **Cross-Server** - Optional multi-server support via Redis
