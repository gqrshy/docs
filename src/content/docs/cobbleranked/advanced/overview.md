## When to Use Advanced Features

### Database

- **SQLite (Default)**: Single server, simple setup
- **MySQL**: Multiple servers sharing data, better for large player bases

### Cross-Server

Required if you want:

- Shared rankings across multiple servers
- Queue from lobby, battle on dedicated server
- Unified leaderboards network-wide
