# Руководство по быстрому старту

---
**CobbleRanked** > **Начало работы** > **Быстрый старт**
---

Запустите рейтинговые сражения менее чем за 5 минут!

## Обзор

Это руководство поможет вам:
1. Настроить первую боевую арену
2. Установить ограничения на покемонов
3. Начать первый сезон
4. Протестировать систему рейтингов

**Требуется времени:** ~5 минут

## Шаг 1: Создание боевой арены

Арены - это места, где проходят рейтинговые сражения.

### Создание арены

1. **Постройте арену** в игре (или используйте существующее место)
2. **Встаньте в точке спавна**, куда вы хотите телепортировать игроков

> **[📸 ТРЕБУЕТСЯ ИЗОБРАЖЕНИЕ: Игрок стоит в месте создания арены (с отображением координат и Yaw/Pitch)]**

3. **Выполните команду:**
   ```
   /rankedadmin arena set <имя_арены>
   ```
   Пример:
   ```
   /rankedadmin arena set main_arena
   ```

4. **Проверьте создание:**
   ```
   /rankedadmin arena list
   ```

### Несколько арен (необязательно)

Создайте несколько арен для разнообразия:

```bash
/rankedadmin arena set volcano_arena
/rankedadmin arena set ice_arena
/rankedadmin arena set forest_arena
```

> **[📸 ТРЕБУЕТСЯ ИЗОБРАЖЕНИЕ: Общая карта с несколькими размещенными аренами (вулкан, лед, лес и т.д.)]**

CobbleRanked будет случайным образом выбирать из доступных арен.

## Шаг 2: Настройка ограничений на покемонов

По умолчанию **все покемоны разрешены**. Давайте запретим некоторых слишком сильных покемонов:

### Редактирование конфигурации черного списка

1. **Откройте файл:** `config/cobbleranked/blacklist.json5`

2. **Запретите легендарных покемонов:**
   ```json5
   {
     "black_list_labels": [
       "legendary",   // Запрещает Mewtwo, Lugia, Ho-Oh, Rayquaza и т.д.
       "mythical"     // Запрещает Mew, Celebi, Jirachi и т.д.
     ]
   }
   ```

3. **Сохраните и перезагрузите:**
   ```
   /rankedadmin reload
   ```

> **[📸 ТРЕБУЕТСЯ ИЗОБРАЖЕНИЕ: Экран с сообщением об успешном выполнении команды перезагрузки]**

### Быстрые пресеты

Выберите соревновательный формат:

## Шаг 3: Начало первого сезона

Сезоны автоматически ротируются на основе вашей конфигурации.

### Проверка текущего сезона

```
/rankedadmin season info
```

Вы должны увидеть:
```
Текущий сезон: Сезон 1
Начался: 2025-10-26
Заканчивается: 2025-11-25 (30 дней)
```

### Настройка длительности сезона

Отредактируйте `config/cobbleranked/config.json5`:

```json5
"ranked_match": {
  "reset_days": 30  // Измените на 7, 14, 30, 60, 90 и т.д.
}
```

Перезагрузка:
```
/rankedadmin reload
```

## Шаг 4: Тестирование системы

Давайте пройдем через полный процесс сражения!

### Вход в очередь

1. **Откройте рейтинговый GUI:**
   ```
   /ranked
   ```
   Или используйте игровой элемент GUI (если настроен)

> **[📸 ТРЕБУЕТСЯ ИЗОБРАЖЕНИЕ: Главный экран GUI рейтингового меню (выбор формата Singles/Doubles)]**

2. **Выберите формат сражения:**
   - Одиночные (1v1)
   - Парные (2v2)

3. **Войдите в очередь**, нажав кнопку очереди

### Начало сражения (требуется 2 игрока)

1. **Второй игрок** входит в очередь того же формата
2. **Подбор противников активируется**, когда два игрока в очереди
3. **Оба игрока телепортируются** на случайную арену
4. **Сражение начинается автоматически**

> **[📸 ТРЕБУЕТСЯ ИЗОБРАЖЕНИЕ: Сцена начала сражения (оба игрока размещены на арене)]**

### После сражения

- **Победитель получает Эло** (по умолчанию: 10-30 очков)
- **Проигравший теряет Эло** (по умолчанию: 10-30 очков)
- **Обновление статистики** (победы, поражения, процент побед)
- **Проверьте таблицу лидеров:** `/ranked` → Таблица лидеров

> **[📸 ТРЕБУЕТСЯ ИЗОБРАЖЕНИЕ: Экран отображения таблицы лидеров (топ игроки и рейтинг Эло)]**

## Шаг 5: Настройка наград (необязательно)

Награждайте своих лучших игроков предметами или командами!

### Редактирование конфигурации наград

Откройте `config/cobbleranked/rewards.json5`:

```json5
{
  "season_rewards": {
    "first_place": {
      "enabled": true,
      "commands": [
        "give %player% minecraft:diamond 64",
        "give %player% cobblemon:master_ball 5"
      ]
    },
    "second_place": {
      "enabled": true,
      "commands": [
        "give %player% minecraft:diamond 32",
        "give %player% cobblemon:master_ball 3"
      ]
    }
  },
  "milestone_rewards": {
    "10_wins": {
      "enabled": true,
      "commands": [
        "give %player% minecraft:emerald 10"
      ]
    }
  }
}
```

Перезагрузка:
```
/rankedadmin reload
```

> **[📸 ТРЕБУЕТСЯ ИЗОБРАЖЕНИЕ: GUI сбора наград (отображение сезонных наград и наград за достижения)]**

## Распространенные проблемы

### Арены не настроены

**Симптом:** Игроки не могут начать сражения

**Решение:**
```
/rankedadmin arena set main_arena
```

### Игроки могут использовать запрещенных покемонов

**Симптом:** Mewtwo разрешен, несмотря на то, что находится в черном списке

**Решение:**
1. Проверьте синтаксис `blacklist.json5` (без завершающих запятых!)
2. Перезагрузите: `/rankedadmin reload`
3. Протестируйте с валидацией GUI `/ranked`

### Эло не обновляется

**Симптом:** Эло остается на 1000 после сражений

**Решение:**
1. Проверьте консоль на наличие ошибок
2. Проверьте соединение с базой данных
3. Проверьте `config.json5` → `eloSystem.mode` (должно быть `LEGACY` или `POKEMON_SHOWDOWN`)

### Сезон уже закончился

**Симптом:** Сообщение «Сезон закончился»

**Решение:**
```
/rankedadmin season rotate
```
Это создаст новый сезон и очистит флаги наград.
