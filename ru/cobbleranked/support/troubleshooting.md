# Устранение неполадок

Распространённые проблемы и их решения для CobbleRanked.

## Проблемы с установкой

### Мод не загружается

**Симптомы:**
- Нет сообщений CobbleRanked в консоли
- Команда `/ranked` не работает
- Мод не появляется в списке модов

**Решения:**

1. **Проверьте установленный Fabric Loader**
2. **Проверьте зависимости:**
   - Fabric API
   - Cobblemon
   - Fabric Language Kotlin
3. **Проверьте логи сервера:**
   ```bash
   cat logs/latest.log | grep CobbleRanked
   ```

### Конфигурационные файлы не генерируются

**Симптомы:**
- Нет папки `config/cobbleranked/`
- Отсутствуют конфигурационные файлы

**Решения:**

1. Запустите сервер хотя бы один раз
2. Проверьте права доступа к файлам
3. Вручную создайте папку:
   ```bash
   mkdir -p config/cobbleranked
   ```

## Проблемы с очередью и матчмейкингом

### Не могу присоединиться к очереди

**Решения:**

1. **Проверьте размер команды:**
   - По умолчанию требуется 6 покемонов
2. **Проверьте, что покемоны живы:**
   - Все покемоны должны иметь HP > 0
3. **Проверьте нарушения черного списка:**
   - Запрещённые покемоны в группе
   - Запрещённые приёмы/способности/предметы

### Очередь занимает вечность

**Решения:**

1. Нужно минимум 2 игрока в очереди
2. Проверьте формат (Одиночные vs Парные)
3. Диапазон Elo расширяется со временем

### Матч найден, но бой не начинается

**Решения:**

1. **Проверьте настроенные арены:**
   ```bash
   /rankedadmin arena list
   ```
2. **Проверьте, что мир арены загружен**
3. **Проверьте ошибки консоли**

## Проблемы с боем

### Бой не начинается после телепортации

**Решения:**

1. Подождите немного (3-секундный обратный отсчёт)
2. Проверьте версию Cobblemon (требуется 1.7.0+)
3. Проверьте, что оба игрока телепортированы

### Elo не обновляется после боя

**Решения:**

1. **Проверьте, закончился ли бой ничьей:**
   - Достигнут лимит ходов (по умолчанию: 100 ходов)
   - Ничьи не изменяют Elo
2. **Проверьте систему Elo включена:**
   ```json5
   {
     "eloSystem": {
       "mode": "POKEMON_SHOWDOWN"
     }
   }
   ```

## Проблемы с конфигурацией

### Конфигурация не перезагружается

**Решения:**

1. Проверьте синтаксис JSON5
2. Убедитесь, что файл сохранён
3. Перезапустите сервер
4. Проверьте консоль на наличие ошибок

### Черный список не работает

**Решения:**

1. Проверьте написание
2. Выполните `/rankedadmin reload`
3. Проверьте синтаксис JSON5

## Проблемы с cross-server

### Игроки не могут видеть друг друга между серверами

**Решения:**

1. **Проверьте подключение MySQL:**
   - Все серверы должны подключаться к одной базе данных MySQL
2. **Проверьте подключение Redis:**
   ```bash
   redis-cli -h host -p port PING
   ```
3. **Проверьте настройку battle_server:**
   - Только ОДИН сервер должен иметь `battle_server: ""`
