# Placeholder API

CobbleRanked интегрируется с [Text Placeholder API](https://placeholders.pb4.eu/) для отображения рейтинговой статистики в голограммах, знаках и других модах Fabric, которые поддерживают плейсхолдеры.

## Обзор

Плейсхолдеры позволяют отображать статистику рейтингов в реальном времени везде, где поддерживается формат Text Placeholder API (`%placeholder%`).

**Распространённые случаи использования:**
- Голограммы таблиц лидеров (моды голограмм на основе Fabric)
- Отображение таблиц (моды таблиц Fabric)
- Форматирование чата (совместимые моды чата Fabric)
- Пользовательские отображения через Text Placeholder API

**Примечание:** Это использует **Text Placeholder API** (мод Fabric), а не PlaceholderAPI (плагин Bukkit/Spigot). Для гибридных серверов (Arclight) поддерживаются оба API.

## Форматы плейсхолдеров

### Все форматы вместе

Отображение лучших игроков по **обоим** форматам Одиночные и Парные вместе (ранжирование по общему Elo).

**Синтаксис:**
```
%cobbleranked_top_<ранг>_<поле>%
```

**Примеры:**
```
%cobbleranked_top_1_name%      → "Notch"
%cobbleranked_top_1_elo%       → "1650"
%cobbleranked_top_2_wins%      → "42"
%cobbleranked_top_3_winrate%   → "75.5"
```

### Только формат Одиночные

```
%cobbleranked_top_singles_<ранг>_<поле>%
```

### Только формат Парные

```
%cobbleranked_top_doubles_<ранг>_<поле>%
```

## Доступные поля

| Поле | Описание | Пример вывода |
|-------|-------------|----------------|
| `name` | Имя пользователя игрока | `"Notch"` |
| `elo` | Текущий рейтинг Elo | `"1650"` |
| `wins` | Всего побед в формате | `"42"` |
| `losses` | Всего поражений в формате | `"15"` |
| `winrate` | Процент побед (1 десятичная) | `"73.7"` |
| `games` | Всего сыгранных игр | `"57"` |

## Поддерживаемые ранги

**Диапазон:** 1-100

Вы можете запросить любой ранг с 1-го места до 100-го места.

## Тестирование плейсхолдеров

```bash
/rankedplaceholder test %cobbleranked_top_1_name%
/rankedplaceholder list
/rankedplaceholder clear
```
