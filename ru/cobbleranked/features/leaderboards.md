# Таблицы лидеров

Просмотр лучших игроков по рейтингу Elo.

## Возможности

- Для конкретного формата (Одиночные, Парные отдельно)
- Обновления в реальном времени после боёв
- Топ-10 отображаемых игроков
- Доступ через GUI или команду

## Просмотр таблиц лидеров

### Через команду

```bash
/leaderboard singles
/leaderboard doubles
```

### Через GUI

1. Откройте рейтинговое меню: `/ranked`
2. Нажмите кнопку "Таблица лидеров"
3. Выберите формат (Одиночные/Парные)

## Отображение таблицы лидеров

Показывает для каждого игрока:
- Позицию в рейтинге (#1, #2 и т.д.)
- Имя игрока
- Текущий рейтинг Elo
- Запись побед/поражений
