# Пользовательская фоновая музыка

Воспроизводите пользовательскую музыку во время рейтинговых боёв.

## Обзор

Использует API `BattleMusicPacket` Cobblemon для отправки музыки игрокам. Не требуются клиентские моды (только музыкальный ресурспак).

**Возможности:**
- Музыка на основе фаз (очередь, выбор команды, бой)
- Музыка боя на основе Elo (более высокие ранги = другая музыка)
- Совместима с Cobblemon 1.6.x и 1.7.x

## Фазы музыки

**Очередь:** Воспроизводится во время ожидания матча
**Выбор команды:** Во время выбора покемонов/обратного отсчёта
**Бой:** Во время активного рейтингового боя (на основе Elo)

## Конфигурация

**Файл:** `config/cobbleranked/config.json5`

```json5
{
  "music": {
    "enabled": true,
    "queueMusic": "cobbleranked:queue_music",
    "teamSelectionMusic": "cobbleranked:team_selection",
    "battleMusic": [
      {
        "minElo": 0,
        "maxElo": 1199,
        "music": "cobblemon:battle_low"
      }
    ]
  }
}
```

См. английскую документацию для создания ресурспаков.
