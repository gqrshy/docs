# Управление сезонами

Временные конкурентные периоды с автоматической ротацией и наградами.

## Обзор

Сезоны - это конкурентные периоды, где игроки соревнуются за позиции в таблице лидеров и награды.

**В конце сезона:**
- Лучшие игроки получают сезонные награды
- Флаги наград сбрасываются
- Новый сезон начинается автоматически
- Статистика игроков (Elo, победы, поражения) **сохраняется**

**Длительность по умолчанию:** 30 дней

## Конфигурация

`config/cobbleranked/config.json5`:

```json5
{
  "ranked_match": {
    "reset_days": 30  // Длительность сезона в днях
  }
}
```

## Автоматическая ротация

Сезоны автоматически ротируются при истечении длительности.

## Ручные команды

```
/rankedadmin endseason
/rankedadmin createseason <дни> [название]
/rankedadmin seasoninfo
```
