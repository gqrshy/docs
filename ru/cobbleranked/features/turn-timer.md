# Таймер ходов

Применяйте временные ограничения на ходы боя для предотвращения затягивания.

## Обзор

У игроков ограниченное время для выбора приёма каждый ход. Если время истекает, автоматически выбирается случайный легальный приём.

**Отображение:**
- Зелёный: > 20 секунд
- Жёлтый: 11-20 секунд
- Красный: ≤ 10 секунд

## Конфигурация

**Файл:** `config/cobbleranked/config.json5`

### Базовая настройка

```json5
"turnTimer": {
  "enabled": true,
  "defaultTimeSeconds": 30
}
```

### Таймеры для конкретных форматов

```json5
"format_timers": {
  "SINGLES": {
    "turn_timeout_seconds": 90
  },
  "DOUBLES": {
    "turn_timeout_seconds": 120
  }
}
```

См. английскую документацию для полных деталей.
