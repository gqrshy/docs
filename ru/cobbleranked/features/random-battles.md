# Случайные бои

---
**CobbleRanked** > **Возможности** > **Случайные бои**
---

Автогенерируемые команды для игроков, которые хотят сражаться без создания команды.

## Обзор

Команды генерируются из предустановленных пулов с фиксированными наборами приёмов, EV и предметами - точно как в Random Battles в Pokemon Showdown.

**Преимущества:**
- Не требуется создание команды
- Идеально для новых игроков
- Фокус на боевых навыках, а не на подготовке

## Форматы

**RANDOM_SINGLES** - 6v6 одиночные с генерированными командами
**RANDOM_DOUBLES** - 6v6 парные с генерированными командами
**RANDOM_3V3** - Быстрые бои 3v3

## Конфигурация

**Файл:** `config/cobbleranked/config.json5`

```json5
"random_battles": {
  "enabled": true,
  "default_pool": "Random OU",  // Название пула из секции random_pools
  "format_pools": {
    "RANDOM_SINGLES": "Random OU",
    "RANDOM_DOUBLES": "Random Doubles OU",
    "RANDOM_3V3": "Random 3v3"
  }
}
```

## Создание пулов покемонов

**Файл:** `config/cobbleranked/random_pools/`

Создайте файлы JSON5 для каждого уровня (например, `random_ou.json5`, `random_ubers.json5`).

### Пример пула

```json5
{
  "pool_name": "Random OU",
  "pokemon": [
    {
      "species": "Garchomp",
      "level": 70,
      "moves": ["Earthquake", "Dragon Claw", "Stone Edge", "Swords Dance"],
      "ability": "Rough Skin",
      "nature": "Jolly",
      "evs": {"hp": 0, "atk": 252, "def": 4, "spa": 0, "spd": 0, "spe": 252},
      "item": "cobblemon:choice_scarf"
    }
  ]
}
```

## Параметры пула

| Поле | Описание | Обязательное |
|-------|-------------|----------|
| `species` | Название покемона | Да |
| `level` | Уровень (1-100) | Да |
| `moves` | 4 приёма | Да |
| `ability` | Название способности | Нет |
| `nature` | Характер | Нет |
| `evs` | Распределение EV | Нет |
| `item` | Удерживаемый предмет | Нет |

## Советы

- Создавайте отдельные пулы для разных уровней (OU, Ubers, LC)
- Используйте EV распределения на основе ролей (Physical Sweeper, Special Wall и т.д.)
- Включайте разнообразные наборы приёмов для каждого покемона
- Тестируйте баланс пула перед развёртыванием

## Устранение неполадок

**Команды не генерируются:** Проверьте, что `random_battles.enabled` установлен в `true` и файлы пулов существуют.

**Отсутствующие покемоны:** Убедитесь, что названия видов соответствуют реестру покемонов Cobblemon.

**Сбои боёв:** Убедитесь, что у всех покемонов есть 4 приёма и валидные способности.
