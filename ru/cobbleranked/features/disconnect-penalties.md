# Штрафы за отключение

Предотвратите злоупотребления, наказывая игроков, которые отключаются во время рейтинговых боёв.

## Обзор

Когда игрок отключается во время рейтингового боя:
- **Счётчик побегов** увеличивается на 1
- **Сдача боя** - Противник автоматически побеждает
- **Штраф Elo** - Теряет Elo, как при обычном поражении
- **Бан очереди** - Временный бан на основе уровня счётчика побегов

## Уровни штрафов

| Счётчик побегов | Бан очереди | Уровень |
|------------|-----------|------|
| 1-5 | 5 минут | ⚠️ Предупреждение |
| 6-10 | 15 минут | 🔶 Серьёзный |
| 11+ | 30 минут | 🔴 Суровый |

## Конфигурация

`config/cobbleranked/config.json5`:

```json5
{
  "competitive": {
    "flee_penalty": {
      "tiers": [
        { "flee_min": 1, "flee_max": 5, "penalty_minutes": 5 },
        { "flee_min": 6, "flee_max": 10, "penalty_minutes": 15 },
        { "flee_min": 11, "flee_max": 999, "penalty_minutes": 30 }
      ]
    }
  }
}
```

См. английскую документацию для полных деталей, включая затухание побегов.
