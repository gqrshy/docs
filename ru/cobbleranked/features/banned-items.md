# Запрещённые предметы инвентаря

Предотвратите присоединение к рейтинговым очередям игроков с определёнными предметами.

## Обзор

Игроки не могут встать в очередь, если у них есть запрещённые предметы в инвентаре. Полезно для блокировки предметов боевых гиммиков, таких как Tera Orbs или Dynamax Bands.

## Конфигурация

**Файл:** `config/cobbleranked/inventory.json5`

```json5
{
  "banned_items": [
    "mega_showdown:tera_orb",
    "mega_showdown:dynamax_band",
    "mega_showdown:mega_bracelet",
    "mega_showdown:z_ring"
  ]
}
```

## Поиск ID предметов

1. Держите предмет в игре
2. Нажмите F3+H (показать расширенные подсказки)
3. Проверьте подсказку для ID (например, `mega_showdown:tera_orb`)

См. английскую документацию для полных деталей.
