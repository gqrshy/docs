# CobbleRanked

Система соревновательных рейтинговых сражений для серверов Cobblemon с подбором противников на основе Эло, сезонными рейтингами и комплексной кастомизацией.

## Возможности

### Основные функции (нулевая конфигурация)
- **Рейтинговый подбор противников** - На основе Эло с динамическим расширением диапазона (±200 начальный, до ±600)
- **Несколько форматов** - Одиночные (1v1), Парные (2v2) с независимыми рейтингами
- **Система сезонов** - Автоматическая ротация каждые 30 дней с наградами для топ-3
- **Таблицы лидеров** - Топ-10 в реальном времени для каждого формата
- **Защита от побегов** - 3-уровневая система штрафов (баны на 5/15/30 минут)

### Дополнительные функции
- **Кросс-сервер** - MySQL/MongoDB + Redis для многосерверных сетей
- **Черный список** - 10 категорий меток + конкретные покемоны/атаки/способности/предметы
- **Пользовательские награды** - Награды в конце сезона (на основе ранга) + достижения (победы/Эло)
- **Боевые арены** - Телепортация на назначенные координаты с пользовательским направлением
- **4 языка** - Английский, Японский, Португальский, Русский

## Быстрый старт

```bash
# 1. Установите зависимости
# 2. Поместите CobbleRanked.jar в mods/
# 3. Запустите сервер
# 4. Протестируйте командой /ranked
```

**Автогенерация:** `config/cobbleranked/` со всеми конфигурационными файлами

**Первый матч:** Игроки начинают с 1000 Эло (режим Pokemon Showdown)

## Система Эло (3 режима)

| Режим | Случай использования | Расчет | Значения по умолчанию |
|------|----------|-------------|----------------|
| **Pokemon Showdown** ✅ | Рекомендуется | K-фактор (32/64) | Старт: 1000, Минимум: 1000 |
| Glicko-2 | 100+ активных игроков | RD + волатильность | Старт: 1500, RD: 350 |
| Legacy | Казуальный | Случайный (10-30) | Н/Д |

<details>
<summary><strong>Подробности Pokemon Showdown</strong></summary>

**Система K-фактора:**
- Новые игроки (< 10 матчей): K=64 (более быстрые изменения)
- Опытные игроки: K=32 (стабильные)

**Пример расчета:**
```
Игрок A (1500 Эло) против Игрока B (1500 Эло)
Ожидаемо: 50% шанс победы
Победа: +16 Эло
Поражение: -16 Эло
```

**Распад Эло:**
- Включен по умолчанию
- -2 Эло в день после 14 дней неактивности
- Выполняется ежедневно в 9:00 UTC

</details>

## Форматы сражений

| Формат | Размер команды | Описание | Рейтинг |
|--------|-----------|-------------|---------|
| **Одиночные** | 3-6 покемонов | Классический 1v1 | Независимый |
| **Парные** | 4-6 покемонов | В стиле VGC 2v2 | Независимый |
| **Командные** | 1-3 каждый | Командные сражения (2v2) | Общий с группой |

**Правила сражений (по умолчанию):**
- ✅ Species Clause (без дубликатов)
- ✅ OHKO Clause (без Fissure/Sheer Cold)
- ✅ Sleep Clause (максимум 1 спящий противник)
- ✅ Evasion Clause
- ✅ Dynamax Clause
- ❌ Item Clause (дубликаты разрешены)
- ❌ Terastal Clause (разрешено)

## Конфигурация по умолчанию

<details>
<summary><strong>Ключевые значения по умолчанию</strong></summary>

```json5
// Сезоны и сражения
"reset_days": 30           // Ежемесячные сезоны
"levelMatch": 0            // Без принудительного уровня
"turn_limit": 100          // Макс. ходов за сражение

// Подбор противников
"initial_range": 200       // ±200 Эло
"expansion_delay": 30      // 30 сек. до расширения
"max_multiplier": 3.0      // Макс. ±600 Эло

// Штрафы за побеги
1-5 побегов:  бан на 5 мин
6-10 побегов: бан на 15 мин
11+ побегов:  бан на 30 мин
Распад: -1 побег за 24 часа

// Эло (Pokemon Showdown)
"initialElo": 1000
"kFactor": 32 (64 для новых игроков)
"provisionalMatches": 10
```

</details>

## Быстрые примеры

### Казуальный сервер (без ограничений)
```json5
// blacklist.json5
{
  "black_list_labels": [],
  "black_list_moves": ["fissure", "sheer_cold", "horn_drill", "guillotine"]
}
```

### Соревновательный сервер (Smogon OU)
```json5
// blacklist.json5
{
  "black_list_labels": ["legendary", "mythical", "restricted", "ultra_beast", "paradox"],
  "black_list_moves": ["baton_pass", "last_respects", "shed_tail"],
  "black_list_ability": ["moody", "shadow_tag", "arena_trap"]
}

// config.json5
{
  "ranked_match": {
    "levelMatch": 50  // Принудительно Lv50
  },
  "eloSystem": {
    "mode": "POKEMON_SHOWDOWN"
  }
}
```

### Формат VGC
```json5
// blacklist.json5
{
  "restricted_label_limits": {
    "restricted": 2  // Макс. 2 ограниченных покемона
  },
  "black_list_labels": ["mythical"]
}

// config.json5
{
  "battle_clauses": {
    "item_clause": true  // Без дублирования предметов
  }
}
```
