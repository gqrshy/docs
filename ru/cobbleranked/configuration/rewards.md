# Система наград

---
**CobbleRanked** > **Конфигурация** > **Награды**
---

Настройте награды за окончание сезона и достижения для рейтинговых игроков.

---

## Расположение файла

`config/cobbleranked/rewards.json5`

---

## Типы наград

### Сезонные награды
Выдаются лучшим игрокам по окончании сезона на основе позиции в таблице лидеров.

### Награды за достижения
Автоматически разблокируются, когда игроки достигают целей (победы, матчи, Elo).

---

## Сезонные награды

```json5
{
  "season_rewards": {
    "singles": {
      "rank_1": {
        "rank_range": "1",
        "display": "&6&l🏆 Награда чемпиона",
        "item": "minecraft:diamond",
        "commands": [
          "give {player} minecraft:diamond 64",
          "eco give {player} 1000"
        ]
      }
    }
  }
}
```

### Поля

| Поле | Обязательное | Описание |
|-------|----------|-------------|
| `rank_range` | ✅ | Требование по рангу (`"1"`, `"2-3"`, `"4-10"`) |
| `display` | ✅ | Отображаемое название (поддерживает цветовые коды `&`) |
| `item` | ✅ | ID предмета Minecraft |
| `commands` | ✅ | Команды для выполнения при получении |
| `lore` | ❌ | Описание предмета (массив) |

**Плейсхолдер:** `{player}` = имя пользователя игрока

## Награды за достижения

```json5
{
  "milestone_rewards": {
    "singles": {
      "wins_10": {
        "type": "WINS",
        "requirement": 10,
        "display": "&e⚡ Первые победы",
        "item": "minecraft:gold_ingot",
        "commands": [
          "give {player} minecraft:gold_ingot 10"
        ]
      }
    }
  }
}
```

### Типы достижений

| Тип | Описание | Пример |
|------|-------------|---------|
| `WINS` | Общее количество побед | 10, 25, 50, 100 |
| `MATCHES` | Всего сыграно матчей | 10, 50, 100 |
| `ELO` | Порог рейтинга Elo | 1200, 1500, 2000 |

```json5
        "item": "minecraft:gold_ingot",
        "commands": ["give {player} minecraft:gold_ingot 32"]
      }
    }
  }
}
```

## Примеры команд

### Выдача предметов
```json5
"commands": ["give {player} minecraft:diamond 64"]
```

### Экономика (требуется плагин экономики)
```json5
"commands": ["eco give {player} 1000"]
```

### Разрешения (требуется LuckPerms)
```json5
"commands": ["lp user {player} permission set ranked.legend true"]
```

### Несколько команд
```json5
"commands": [
  "give {player} minecraft:diamond 64",
  "eco give {player} 1000",
  "lp user {player} permission set ranked.legend true",
  "broadcast &6{player} &eстал легендой!"
]
```

| Код | Цвет | Код | Формат |
|------|-------|------|--------|
| `&0` | Чёрный | `&l` | Жирный |
| `&1` | Тёмно-синий | `&m` | Зачёркнутый |
| `&2` | Тёмно-зелёный | `&n` | Подчёркнутый |
| `&3` | Тёмно-голубой | `&o` | Курсив |
| `&4` | Тёмно-красный | `&r` | Сброс |
| `&5` | Тёмно-фиолетовый |
| `&6` | Золотой |
| `&7` | Серый |
| `&8` | Тёмно-серый |
| `&9` | Синий |
| `&a` | Зелёный |
| `&b` | Голубой |
| `&c` | Красный |
| `&d` | Светло-фиолетовый |

## Награды для конкретных форматов

Каждый формат имеет независимые награды:

```json5
{
  "season_rewards": {
    "singles": { /* Награды одиночных */ },
    "doubles": { /* Награды парных */ }
  }
}
```

Игроки могут получать награды в **обоих** форматах.

## Перезагрузка

После редактирования `rewards.json5`:

```
/rankedadmin reload
```

---

## Устранение неполадок

**Награды не появляются?**
- Проверьте синтаксис `rank_range`
- Убедитесь, что сезон закончился: `/rankedadmin season info`
- Перезагрузите конфигурацию: `/rankedadmin reload`

**Команды не выполняются?**
- Сначала протестируйте команду вручную
- Проверьте написание плейсхолдера `{player}`
- Убедитесь, что требуемые плагины установлены

**Цветовые коды не работают?**
- Используйте `&`, а не `§`
- Проверьте экранирование JSON
