## Формат конфигурации

CobbleRanked использует **формат JSON5** для всех конфигурационных файлов. JSON5:
- ✅ Легко читать и редактировать
- ✅ Поддерживает комментарии (`//` или `/* */`)
- ✅ Разрешает завершающие запятые
- ✅ Более гибкий, чем стандартный JSON

## Расположение конфигурационных файлов

Все конфигурационные файлы находятся в: `config/cobbleranked/`

```
config/cobbleranked/
├── config.json5         # Основная конфигурация
├── blacklist.json5      # Запреты покемонов/приёмов/способностей
├── arenas.json5         # Расположения боевых арен
├── rewards.json5        # Награды за сезоны и достижения
├── gui/
│   ├── gui-enUs.json5   # Английский GUI
│   ├── gui-jaJp.json5   # Японский GUI
│   ├── gui-ptBr.json5   # Португальский GUI
│   └── gui-ruRu.json5   # Русский GUI
└── language/
    ├── en-Us.json5      # Английские сообщения
    ├── ja-Jp.json5      # Японские сообщения
    ├── pt-Br.json5      # Португальские сообщения
    └── ru-Ru.json5      # Русские сообщения
```

## Первоначальная настройка

1. **Запустите сервер** - Конфигурационные файлы будут созданы автоматически
2. **Остановите сервер**
3. **Отредактируйте файлы** с помощью любого текстового редактора (VS Code, Notepad++ и т.д.)
4. **Перезапустите сервер** - Изменения будут применены

## Редактирование файлов JSON5

### Основные правила

1. **Двойные кавычки для строк**: `"legendary"` (одинарные кавычки также работают: `'legendary'`)
2. **Завершающие запятые разрешены**: `["A", "B",]` допустимо
3. **Комментарии разрешены**: Используйте `//` или `/* */`
4. **Пустые списки**: `[]`
5. **Пустые объекты**: `{}`

### Пример: Запрет покемонов

```json5
{
  // Запретить легендарных покемонов
  "black_list_pokemon": [
    "Mewtwo",
    "Lugia",
    "Rayquaza",  // Завершающая запятая допустима в JSON5
  ]
}
```

### Возможности JSON5

✅ **Комментарии:**
```json5
{
  // Это комментарий
  "black_list_pokemon": ["Mewtwo"], // Комментарий в конце строки

  /* Многострочный
     комментарий */
  "black_list_labels": ["legendary"]
}
```

✅ **Завершающие запятые:**
```json5
{
  "black_list_pokemon": [
    "Mewtwo",
    "Lugia",  // Завершающая запятая допустима
  ],  // Здесь тоже допустима
}
```

## Перезагрузка конфигурации

Используйте команду `/rankedadmin reload` для перезагрузки конфигурации без перезапуска:

```
/rankedadmin reload
```
