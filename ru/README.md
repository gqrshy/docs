## 📦 Моды

### CobbleRanked

Комплексная система рейтинговых сражений для серверов Cobblemon.

**Возможности:**
- 🏆 Система подбора противников на основе рейтинга Эло
- ⚔️ Несколько форматов сражений (одиночные, парные)
- 📊 Сезонные таблицы лидеров с автоматической ротацией
- 🚫 Настраиваемые ограничения покемонов/атак/способностей
- 🌍 Кросс-серверная поддержка (MySQL + Redis)
