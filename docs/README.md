# CobbleRanked Documentation

Welcome to **CobbleRanked** - the ultimate ranked competitive PvP system for Cobblemon servers!

## What is CobbleRanked?

CobbleRanked brings competitive Pokemon battling to your Cobblemon server with a fully-featured ranked matchmaking system. Battle other players, climb the leaderboards, earn rewards, and prove you're the best trainer on your server!

## Key Features

- **Two Battle Formats**: Singles (1v1) and Doubles (2v2) with separate rankings
- **Smart Matchmaking**: Dynamic Elo-based system finds fair matches quickly
- **Season System**: Compete in ranked seasons with automatic rotation and end-of-season rewards
- **Customizable Rules**: Ban Pokemon, moves, abilities, and items per format
- **Reward System**: Earn rewards for reaching milestones and finishing in top ranks
- **Cross-Server Support**: Optional multi-server setup for larger networks
- **Beautiful GUIs**: Intuitive menus with custom sounds and animations
